### AccessDiag
**Making educational content accessible across languages and abilities**

| | |
|---|---|
| **Problem** | Students with visual impairments or language barriers struggle with diagram-heavy NCERT content |
| **What I built** | A multimodal AI pipeline that takes a diagram or spoken question and returns a grade-appropriate spoken explanation in English, Hindi, or Marathi |
