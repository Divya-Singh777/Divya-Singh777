## 🚧 What I'm building

### AccessDiag
Multimodal AI pipeline for accessible education in Indian languages.

A student with a visual impairment or a language barrier shouldn't get left behind by a diagram in a textbook.  
This system takes NCERT diagrams + vernacular speech input and returns grade-calibrated spoken explanations in English, Hindi, and Marathi.

Under the hood:
- ViT-B/16 for diagram understanding + Whisper ASR for speech input
- 5-agent LLM-driven pipeline: Intent Router → Diagram RAG → Explanation → Grade-Level Context → TTS
- Ensemble classifier: BiLSTM (92.1% acc) + CNN spectrogram (94.6%) → **95.2% ensemble accuracy**
- GAN-based augmentation to handle data scarcity
- IndicTrans2 + FastSpeech 2 + HiFi-GAN for multilingual TTS output

> Built for students who don't have the luxury of a tutor explaining things twice.

XGBoost + Isolation Forest for anomaly detection.  
LSTM for time-series patterns.  
FastAPI for low-latency inference with alert and escalation integration.

### Fake Review Detection with Explainable AI
NLP pipeline for e-commerce review integrity.

The part I found most interesting wasn't the classification — it was making the model explain itself.  
SHAP attribution means you see *which words* pushed the prediction, not just a confidence score.

Deployed as a live Streamlit app.
